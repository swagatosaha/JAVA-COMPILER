## Packages
@monaco-editor/react | Professional code editing experience (VS Code-like)
react-dropzone | Drag and drop file uploads for the decompiler
framer-motion | Smooth UI transitions and animations
date-fns | Formatting timestamps for snippet history
clsx | Conditional class merging
tailwind-merge | Tailwind class merging utility

## Notes
- Java Compiler expects POST /api/compiler/compile with { code: string }
- Decompiler expects POST /api/decompiler/upload with multipart/form-data
- Dark mode is the primary aesthetic (terminal style)
- Monaco Editor will handle Java syntax highlighting
